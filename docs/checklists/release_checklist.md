# Release Checklist

- Application starts successfully.
- `pytest -q` is green.
- `ruff check .` is green.
- README is current.
- Public disclaimer is visible.
- Portfolio assets are listed.
- Demo data works.
- Admin dashboard opens and renders cleanly.
- Known issues and boundaries are documented.
- Release note is current.
- If live validation was run, cleanup artifact and cleanup audit were completed.

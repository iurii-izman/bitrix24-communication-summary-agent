# GitHub Public Repo Checklist

- repository description added
- topics added
- social preview prepared
- `README.md` renders cleanly
- license visible on repository home
- issue forms enabled and tested
- pull request template enabled
- security policy visible
- support file visible
- CI workflow enabled
- Dependabot enabled
- branch protection or ruleset applied to `main`
- code owners enabled only after real owner mapping is configured
- no secrets, local databases, or runtime artifacts pushed
